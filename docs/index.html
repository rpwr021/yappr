Yappr is a push-to-talk dictation & voice-chat app that lives in your macOS menu bar. Speech recognition, the language model, and text-to-speech all run on your machine — nothing is uploaded, no account, no telemetry. Hold Right Option, speak, release. Your words land in whatever app you're using.

01

The gesture

⌥ HOLD

Dictate anywhere

Hold Right Option and speak. Release, and the transcription is pasted straight into the focused app — editor, terminal, chat box.

⌘ + ⌥

Ask out loud

Hold Command + Right Option to ask a spoken question. Yappr answers concisely and reads it back through local text-to-speech.

◍ VAD

Silence-aware

A local Silero voice-activity gate trims dead air before transcription, so half-second throat-clears don't become phantom text.

⌕ SEARCH

Grounded answers

For time-sensitive questions the model can call a web-search tool, then summarizes real results instead of guessing.

02

Built for trust

A.

Nothing leaves

Audio, transcription, and inference happen locally via llama.cpp. No request ever carries your voice off the machine.

B.

Audio-native model

One Gemma audio model both hears and answers, so dictation stays a fast single round-trip — no pipeline of cloud services.

C.

Lives in the bar

A tiny menu-bar duck shows state at a glance — listening, transcribing, speaking — and switches mic, model, and language.

D.

Instant paste

Release the key and text appears where your cursor is. No window to switch to, no copy-paste dance.

E.

Native Rust app

Written in Rust — a single fast, memory-safe binary, no Electron. Tuned through a plain config.ini: model, voice, language, VAD, search.

F.

One-line install

Homebrew tap or a curl script. The engine installs itself; first launch fetches the model with on-screen progress.

03

Get Yappr

Two commands. That's it.

macOS · Apple Silicon · requires Input Monitoring + Microphone permission

homebrew
$ brew install rpwr021/yappr/yappr
curl
$ curl -fsSL https://raw.githubusercontent.com/rpwr021/yappr/main/install.sh | bash

Homebrew pulls in the llama.cpp engine automatically. First launch downloads the speech model (a few minutes) — the menu-bar duck shows the progress. Update later with brew upgrade --cask yappr.
